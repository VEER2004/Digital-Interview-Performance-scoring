    ↓

    ↓

    ├─ Eye contact scoring
    └─ Facial expression analysis
    ↓
Phase 2: Speech Analysis (Optional)
    ├─ Audio quality
    ├─ Speaking pace
    └─ Sentiment analysis
    ↓
Phase 3: Body Language Analysis
    ├─ Motion detection
    ├─ Posture analysis
    └─ Gesture scoring
    ↓
Phase 4: Unified Scoring
    ├─ Weighted combination
    ├─ Consistency check
    └─ Overall AI score
    ↓
ML Model Prediction
    ├─ Linear Regression model
    ├─ 25+ input features
    └─ Final performance score (0-30)
    ↓
Categorization & Output
    ├─ Excellent (≥20)
    ├─ Good (≥12)
    └─ Needs Improvement (<12)
